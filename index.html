土石流偵測預警

Mudslide Prediction System



了解更多

發想

土石流在台灣是一種常見的自然災害，我們希望能藉由這學期所學的知識，幫助我們建立一套有助於偵測/預警土石流災害的系統

Our main goal is to construct a image recognition model (using CNN), which can distinguish whether a mudslide is happening

利用政府資料

Open Data

利用政府土石流觀測站的影像，訓練模型以檢測土石流的發生

深度學習模型

Deep Learning Model

使用NasNet捲積類神經網路架構，再圖像辨識上擁有高精確度

及時偵測預警

Instant Detection

結合政府觀測站及時影像，系統有能力及時判斷土石流是否發生

資料來源

為了訓練我們的深度學習模型，我們從政府的土石流防治網站，以及google搜尋取得相關圖片進行訓練

To train and evaluate our model, we collect images from both government's website and google search

範例影像-政府土石流防災網

範例影像-Google圖片

模型架構

我們使用Google brain所提出的NasNet模型，用強化學習的方式自動產生網絡結構

We use NasNet model from google brain, which uses reinforcement learning to find out the best architecture
